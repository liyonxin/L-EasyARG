# L-EasyARG

A comprehensive, user-friendly tool for analyzing antibiotic resistance genes (ARGs) from metagenomic sequencing data.

## Features

- **Complete Analysis Pipeline**: From raw reads to annotated ARGs
- **Multiple Database Support**: CARD, PLSDB, MGE databases
- **Taxonomic Classification**: Centrifuge-based microbial classification
- **Visualization**: Multiple plotting options for result interpretation
- **User-friendly**: Simple command-line interface with sensible defaults
